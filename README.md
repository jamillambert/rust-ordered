Arbitrary Ordering
==================

Provides a wrapper for types that can technically implement `PartialOrd`/`Ord`
but for semantic reasons it is nonsensical.

## Minimum Supported Rust Version (MSRV)

The crate MSRV is Rust v1.56.1
